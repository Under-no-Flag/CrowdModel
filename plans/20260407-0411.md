### 任务细化
#### 1. 明确 5 组实验的定义
- [x] **实验组 1：模型机制验证**
  - 研究问题：`M(x)`、`U(x)`、多阶段分流是否分别带来可识别的机制变化
  - 计划对比：
    - 基线：`case1_baseline`
    - 仅几何引导：待补 `M-only` case
    - 仅单向约束：待补 `U-only` case
    - 双控：以中通道为主的 `M+U` case
  - 自变量：是否启用 `M`、是否启用 `U`、是否启用阶段分流
  - 因变量：平均密度、峰值密度、通道流量占比、速度场间断度
  - 控制变量：几何、初始密度、网格、时间步、速度函数
- [x] **实验组 2：主场景策略对比**
  - 研究问题：不同通道管控策略中，哪种方案在效率、安全与均衡性上更优
  - 计划对比：
    - `case1_baseline`
    - `case2_middle_guided`
    - `case3_top_guided`
    - `case4_bottom_guided`
  - 自变量：受控通道位置、单向约束布置、几何引导方向
  - 因变量：`J1/J2/J5` 对应统计量、峰值密度、累计通过量
  - 控制变量：相同场景、相同初值、相同参数
- [x] **实验组 3：多阶段分流行为验证**
  - 研究问题：多阶段行为链和固定概率分流是否会改变出口负载与阶段密度演化
  - 计划对比：
    - 单阶段近似 case
    - `case5_multistage_tour`
    - 必要时补一个“无分流/单出口”的对照 case
  - 自变量：是否分阶段、是否分流、分流概率
  - 因变量：各阶段群体质量、出口分担、总旅行时间、高密度暴露
  - 控制变量：共享拥堵函数、相同几何、相同起始人群
- [x] **实验组 4：最优控制参数求解**
  - 研究问题：在给定目标函数下，哪些管控参数组合最优
  - 计划变量：
    - 离散变量：受控通道/策略编号
    - 连续变量：各向异性比值 `eta`
    - 连续变量：固定分流概率 `p`
  - 计划目标：最小化 `J = λ1 J1 + λ2 J2 + λ5 J5`
  - 第一版方法：有限候选搜索 + 粗网格搜索
  - 暂不实现：完整 SA-HBO
- [ ] **实验组 5：鲁棒性/扩展预留**
  - 研究问题：主结论在需求扰动、偏好变化、局部容量下降下是否仍成立
  - Day 1 只完成设计，不进入本周正式实现
  - 预留变量：入流强度、分流概率扰动、通道容量折减、局部禁行

#### 2. 建立实验假设矩阵
- [ ] 为每组实验至少写出 1-3 条可检验假设
- [ ] 假设写法统一为：

  - `H2`：单向约束会减少逆向冲突，但可能引起局部堆积转移
  - `H3`：多阶段分流会显著改变出口负载分配
- [ ] 每条假设都配套：

  - 判定标准
  - 需要的代码能力
- [ ] 为最优控制参数求解补充：

  - `H_opt2`：最优结果会随 `λ1/λ2/λ5` 权重变化而迁移
  - `H_opt3`：离散策略变量与连续参数变量都对目标函数有显著影响
