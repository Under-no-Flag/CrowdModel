# 本周工作汇报

**汇报周期**：2026年5月14日 - 2026年5月18日

## 一、本周主要工作

本周主要工作是围绕 G4 实验中暴露出来的问题，重新梳理模型控制变量和优化算法路线。整体上，先完成了 G4 的多方法对比实验，然后根据实验结果分析原有 SA-HBO 效果不理想的原因，进一步调研文献，最后完成了 model v2 和 optimization v2 的设计，并开始实现新的 G5 优化实验。

---

## 二、G4 对比实验与问题发现

首先实现并整理了 G4 实验的多方法对比，主要比较了 baseline、random search、grid search、pure SA、SA-HBO no proxy 和 SA-HBO 等方法。

实验结果显示，当前 SA-HBO 虽然比 baseline 和 grid search 有改进，但没有超过 random search 和 pure SA。这个结果说明，之前设计的结构感知优化框架并不是完全无效，但它目前还不能支撑“明显优于传统方法”的结论。

针对这个问题，做了进一步分析，主要发现三点：

1. 安全指标 J2 原来采用硬阈值形式，数值明显小于 J1 和 J5，在加权目标里几乎不起主要作用。
2. 连续变量 eta 对目标函数的影响很弱，优化过程中几乎没有真正发挥作用。
3. 当前 SA-HBO 的离散搜索范围偏局部，代理筛选也可能没有保留真正好的候选。

因此，认为问题的关键不只是优化算法本身，而是原来的控制变量 eta 可能没有选在最敏感、最可实施的位置上。

---

## 三、文献调研与模型改造思路

在确认 eta 作用较弱之后，调研了连续介质人群模型中关于通道入口流入速率控制的相关文献，并撰写了调研报告。

通过调研，发现近年的相关研究更倾向于把入口流量、节点供需关系、分流比例和设施级控制作为主要管控变量，而不是继续微调域内的连续张量参数。也就是说，真正有效的控制往往发生在入口、边界、节点或设施层面。

基于这个判断，对模型改造形成了新的思路：

- 不再把 eta 作为主要优化变量；
- 保留 eta 对应的几何引导作用，但只把它作为固定结构参数；
- 将主要控制变量改成通道方向配置 s 和内部入口通行速率 q；
- 入口限流不应让人群“消失”，而应表现为入口外等待或排队；
- 优化目标中需要加入入口等待惩罚，避免算法通过过度限流得到不可实施的方案。

---

## 四、Model v2 的设计

根据文献调研结果，完成了 model v2 的建模思路。新版模型把原来的优化变量：

```text
z = (s, eta)
```

改为：

```text
z = (s, q)
```

其中，s 表示通道方向规则，例如单向、反向、双向或关闭；q 表示每条通道入口在不同时间段允许通过的人流速率。eta 不再参与优化，而是固定为 eta0，只保留为通道内部几何引导机制。

还明确了方向和入口容量之间的一致性关系。例如，若某条通道只允许东向通行，那么反向入口容量必须为 0；若通道关闭，则两个方向的容量都为 0；若通道双向开放，则两个方向共享总容量约束。

此外，在新版模型中加入了入口等待指标 J_B 和容量平滑性指标 J_R。前者用于防止过度限流，后者用于避免容量控制频繁跳变，使方案更接近现实管理动作。

---

## 五、Optimization v2 的设计

在模型改造之后，又调研了混合整数 PDE 约束优化、混合变量贝叶斯优化和无导数优化等相关方法，并据此设计了 optimization v2。

对当前问题的判断是：它不是普通连续优化问题，而是一个混合离散-连续、动态、非光滑、昂贵仿真驱动的黑盒优化问题。方向配置 s 是离散变量，入口速率 q 是连续变量，每次评价都需要运行完整的人群仿真，因此不适合直接套用普通梯度优化或通用 MINLP 求解器。

最终采用的思路是：

1. 外层先生成和筛选方向配置 s；
2. 内层在固定 s 的情况下优化入口速率 q；
3. 使用低保真仿真先筛掉明显较差的候选；
4. 对较好的候选进行局部抛光；
5. 最终只用高保真仿真结果作为排序和结论依据；
6. 补充 random search、only-s、only-q、no-DFO、no-JB 等对比和消融实验。

这个设计的重点是让优化过程既符合模型结构，又能控制计算成本。

## 六、代码实现与实验进展

在理论设计完成后，开始实现新的 G5 优化算法和对应实验。

目前已经完成的主要内容包括：

- 实现新的控制变量 z=(s,q)；
- 实现入口容量的可行映射，保证方向规则和容量约束一致；
- 实现 no-cap 参考，用来估计各入口自然尝试流量；
- 实现低保真筛选、中保真优化和高保真复验流程；
- 实现随机搜索、only-s、only-q、no-DFO、no-JB 等对比实验；
- 增加统一实验矩阵，并支持并行运行多组实验；
- 完成小规模 smoke 测试，确认七组实验都可以正常生成结果。

当前全量 G5 实验已经开始运行。已有部分实验完成，其中 random search 暂时取得了当前已完成候选中的最好结果。不过由于完整矩阵还没有全部结束，所以现在还不能给出最终结论。

## 七、本周阶段性结论

本周最重要的收获是，没有继续强行优化原来的 eta，而是通过实验负结果发现了模型控制变量的问题。

G4 对比实验说明，当前 SA-HBO 的效果不够好；进一步分析表明，根源在于 eta 不是一个足够强、足够可实施的主动控制变量。文献调研也支持这个判断：入口流入速率、节点分流和设施级限流更接近现实管控动作，也更符合连续介质人群模型中的有效控制位置。

因此，完成了从旧模型 z=(s,eta) 到新模型 z=(s,q) 的转向，并进一步设计和实现了新的 G5 优化实验框架。下一步需要等待全量实验结束，再根据结果判断新模型和新算法是否能真正优于基线方法。
